### Jump Statements

#### break 
Keluar dari loop atau switch.
#### continue 
Melewatkan iterasi saat ini dalam loop dan melanjutkan ke iterasi berikutnya.

**Contoh:**

```js
for (let i = 0; i < 10; i++) {
  if (i === 5) {
     break; // Keluar dari loop ketika i sama dengan 5
  }
  console.log(i);
}
```
